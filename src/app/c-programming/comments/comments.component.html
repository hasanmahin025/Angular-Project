C Comments

Let's Start C Comments

Comments are used to explain code and it's make code readable and understandable for us

There are Two Types of Comments

1.Single Line
2.Multiline Comments

Lets Start with Single Line Comments

1.Single Line Comments

For Example

printf("Hello world") // This is single line comments

Let's Start Multiple Line Comments

2.Multiple Line Comments

Multi-line comments start with /* and ends with */. Any text between /* and */ will be ignored by the compiler:

For Example

printf("hello world\n");

/*This is
Multi-line
