C OutPut

Let's start with C output

To see output in c we can use printf() function

For example


  #include <stdio.h>
  int main() {
      printf("Hello World");

      return 0;
  }

when we print a line like text then we can use Double qoutes mark

You can see the example.

when you print a text without double qoutes you get a error

You can use many printf function as you want

for example


  #include <stdio.h>
  int main() {
      printf("Hello World");
      printf("hello");
      printf("hi");
      return 0;
  }

For a new line you can use '\n' chacrater

for Example


        #include <stdio.h >
        int main() {
        printf("hello world\n");
        return 0;
        }

Also you can use multiple newline in a single printf() function

for example


        #include <stdio.h >
        int main() {
        printf("hello\nworld\n");
        return 0;
        }

You can Run this in your codeblocks IDE then see what happens
