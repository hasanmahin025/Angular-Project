C Variables

Let's Start with C Variables

In C programming, variables are named,
storage locations in memory,
used to store data that can be changed during program execution.

So We can say that A variable in C
is a container for storing data values such as integers,
floating-point numbers, or characters.

Rules for Naming Variables:

1.Must begin with a letter (A–Z or a–z) or an underscore (_)
2.Cannot start with a digit
3. Can contain letters, digits, and underscores
4. No special characters or spaces
5.Cannot use C keywords (like int, return, float)

Let's see a example


  #include <stdio.h>
  int main() {
    int age = 20;
    float height = 5.9;
    char initial = 'J';

    printf("Age: %d\n", age);
    printf("Height: %.1f\n", height);
    printf("Initial: %c\n", initial);

    return 0;
  }
