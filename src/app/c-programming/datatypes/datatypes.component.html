C Data Types

Let's Start with C Data Types

C provides several built-in data types that define the type of data
a variable can hold.These types fall into the following categories:

Different Types of Data Types in C

1.Primary Data Types

Data Type Description Size (Typical) Example Values
int Stores whole numbers 4 bytes 10, -5, 0
float Stores single-precision decimal numbers 4 bytes 3.14, -0.5
double Stores double-precision decimal numbers 8 bytes 3.14159, -2.5e10
char Stores a single character 1 byte 'a', 'Z', '1'
void Indicates absence of value 0 bytes void main()

For Example


  #include <stdio.h>
  int main() {
    int id = 20;
    float cgpa = 3.43;
    char Grade = 'A';
    double Result = 3.56;

    printf("id: %d\n", id);
    printf("cgpa: %.1f\n", cgpa);
    printf("Grade: %c\n", Grade);
    printf("Result %lf\n" , Result);

    return 0;
  }

Here is a table of the Basic Format Specifiers in C, commonly used with functions like printf() and scanf():

Specifier Data Type Description Example
%d int Signed decimal integer printf("%d", 10);
%u unsigned int Unsigned decimal integer printf("%u", 100);
%f float Decimal floating point printf("%f", 3.14);
%lf double Double-precision float printf("%lf", 3.141592);
%c char Single character printf("%c", 'A');
%s char[] String of characters printf("%s", "Hello");
%x int Hexadecimal (lowercase) printf("%x", 255);
%X int Hexadecimal (uppercase) printf("%X", 255);
%o int Octal number printf("%o", 10);
%% N/A Prints a percent sign printf("%%");
