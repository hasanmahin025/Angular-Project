C input

Let's Start With C Input

1. scanf() – Formatted Input

Used to read input from the standard input (keyboard).

scanf("format_specifier", &variable);

Example:

int age;
scanf("%d", &age);

Common Format Specifiers:

SpecifierTypeExample
%dintscanf("%d", &x);
%ffloatscanf("%f", &f);
%lfdoublescanf("%lf", &d);
%ccharscanf(" %c", &ch); (space skips newline)
%sstringscanf("%s", str);

2. getchar() – Read a Single Character

char ch = getchar();

Example:

char ch;
printf("Enter a character: ");
ch = getchar();

3. gets() – Deprecated

Avoid using gets() – it is unsafe and deprecated.

Instead, use fgets():

fgets(str, size, stdin);

Example:

char name[50];
printf("Enter your full name: ");
fgets(name, 50, stdin);
